LinksDetektorOptions

enum LinksDetektorOptions : Enum<LinksDetektorOptions>

Options for configuring URL detection behavior.

This enum is implemented as a bit mask, allowing multiple options to be combined. Each option represents a specific detection behavior that can be enabled or disabled. Options can be combined using bitwise operations through the hasFlag method.
