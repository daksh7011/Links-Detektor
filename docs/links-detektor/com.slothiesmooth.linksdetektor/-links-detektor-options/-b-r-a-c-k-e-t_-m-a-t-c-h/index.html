BRACKET_MATCH

BRACKET_MATCH

Enables bracket matching for URLs.

This option handles URLs enclosed in various bracket types: (), {}, and []. Similar to quote matching, the brackets themselves are excluded from the URL.

Bit value: 4 (00000100)
